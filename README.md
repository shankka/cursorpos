#CURSOR POSITION#

C#写的windows程序，作用为获取当前鼠标光标所在位置。

getpos.exe为可执行程序,你可直接使用，也可自行编译。编译命令:

<pre>
<code>
C:\Windows\Microsoft.NET\Framework\...(版本)\csc.exe /t:winexe getpos.cs
</code>
</pre>
